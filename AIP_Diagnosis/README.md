# 自己免疫性膵炎 診断基準計算ツール

自己免疫性膵炎の診断基準をクリック入力で確認する単体HTMLツールです。

## 内容

- 自己免疫性膵炎診断基準2018: 確診、準確診、疑診を判定
- ICDC: 1型/2型AIPの確診、準確診、AIP疑診を判定
- 病理組織学的4所見: 4所見中3件以上かを判定

## 使い方

`index.html` をブラウザで開くだけで使用できます。

## noteで提示する方法

note本文にはJavaScript付きHTMLを直接動かす形では載せにくいため、以下の形を推奨します。

1. `index.html` をNetlify、GitHub Pages、Vercelなどで公開する
2. 公開URLをnote本文に貼る
3. note本文には概要、注意事項、リンクを掲載する

## Netlify Dropで公開する手順

1. https://app.netlify.com/drop を開く
2. このフォルダ全体、または `index.html` をドラッグ&ドロップする
3. 表示されたURLをコピーする
4. `note_post_draft.md` の `https://公開URLをここに貼る` を置き換える
5. note本文に貼り付ける

## 注意

このツールは診断補助・教育目的です。膵癌・胆管癌など悪性疾患の除外、ステロイド診断の適否、最終診断は専門医の臨床判断で行ってください。
